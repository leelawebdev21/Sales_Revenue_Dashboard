# Sales & Revenue Dashboard
### Excel + Power BI | Beginner Portfolio Project

A complete Sales & Revenue analysis project built with **Microsoft Excel** and **Power BI Desktop** — great for learning data analytics basics and adding to your GitHub portfolio.

---

## Project Structure

```
sales-revenue-dashboard/
├── Sales_Revenue_Dashboard.xlsx   ← Main Excel file (5 sheets, 892 formulas)
├── POWERBI_GUIDE.md               ← Step-by-step Power BI build guide
└── README.md
```

---

## What's Inside the Excel File

| Sheet | What It Contains |
|-------|-----------------|
| Sales Data | 230+ raw transactions — orders, products, regions, revenue |
| Monthly Summary | Revenue & orders rolled up by month using SUMIF/COUNTIF |
| Category Analysis | Revenue breakdown across 5 product categories |
| Region Analysis | North / South / East / West performance comparison |
| KPI Summary | Key metrics (best for Power BI import) |

---

## Key Business Questions Answered

- Which month had the highest revenue?
- Which product category drives the most sales?
- How much revenue is lost to discounts?
- Which region performs best?
- What is the average order value?

---

## How to Use

### Excel
1. Open `Sales_Revenue_Dashboard.xlsx` in Microsoft Excel
2. All formulas auto-calculate — no setup needed
3. Explore each sheet tab at the bottom

### Power BI
1. Install **Power BI Desktop** (free) from powerbi.microsoft.com
2. Follow the step-by-step `POWERBI_GUIDE.md`
3. Connect to this Excel file and build 4 dashboard pages
